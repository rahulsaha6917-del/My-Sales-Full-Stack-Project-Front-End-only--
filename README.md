# 🛒 SalesSavvy – Full Stack E-Commerce Project

SalesSavvy is a full-stack e-commerce web application where users can browse products, view categories, and shop online.
The project is built using **React (Vite)** for the frontend and **Spring Boot + MySQL** for the backend.

---

## 🚀 Features

### 👤 User Features

* Browse products by categories
* View product images, name, and price
* Search products
* Add products to cart
* User authentication (Login / Register)
* Responsive UI

### 🛠 Admin Features

* Manage products
* Manage categories
* View orders
* Manage product images
* Database integration

---
